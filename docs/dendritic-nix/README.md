# Dendritic Nix Documentation Suite

This is the deep, multi-file documentation set for **Dendritic Nix** (sometimes misspelled as _dendretic_ / _dendretix_).

Canonical naming:

- **Dendritic Pattern** - the architecture pattern
- **Den** - aspect-oriented framework built around/alongside dendritic ideas
- **Dendrix** - community distribution of reusable dendritic modules
